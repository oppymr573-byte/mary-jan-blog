# Mary Jan Blog 🚀

Welcome to my personal blog project! This is a fully functional web application built with *HTML, CSS, and JavaScript, connected to **Firebase Firestore* for real-time data management.

## ✨ Features
- *Real-time Posting:* Add blog posts that appear instantly without refreshing.
- *Cloud Database:* All posts are securely stored in the Google Firebase Cloud.
- *Delete Functionality:* Easily manage your feed by deleting posts.
- *Responsive Design:* Works perfectly on both desktop and mobile.

## 🛠️ Tech Stack
- *Frontend:* HTML5, CSS3, JavaScript (ES6)
- *Backend:* Firebase Firestore (NoSQL Database)
- *Deployment:* GitHub Pages
